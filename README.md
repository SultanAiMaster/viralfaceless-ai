## 🎯 What Is It?

ViralFaceless AI is a comprehensive automation system that creates, generates, and uploads faceless YouTube videos to 10 different channels every single day.

### Key Features

✅ **10 Automated Channels** - Each targeting a viral niche
✅ **30 Videos Daily** - 3 videos per channel, optimized for virality
✅ **24/7 Operation** - Runs continuously on cloud servers
✅ **AI-Powered** - Scripts, voiceovers, visuals all generated by AI
✅ **Smart Scheduling** - Uploads at optimal times for each channel
✅ **Auto-Optimization** - Analytics-driven content improvement

## 🛠️ Tech Stack

### Core Technologies
- **Backend:** Node.js + TypeScript + Express
- **Queue System:** BullMQ (Redis-based job scheduling)
- **Database:** MongoDB (analytics, content storage)
- **Cache:** Redis (queue management)

### AI Integration
- **OpenAI GPT-4:** Script writing, metadata generation
- **ElevenLabs:** AI voiceover (professional quality)
- **MuleRouter:** AI video generation
- **Midjourney:** Thumbnail creation

### YouTube Automation
- **YouTube Data API v3:** Upload, manage, analytics
- **Google OAuth2:** Multi-channel authentication

### Video Processing
- **FFmpeg:** Video editing & compilation
- **Stock APIs:** Pexels, Pixabay (footage)

## 🚀 Quick Start

### Prerequisites
- Node.js 22+
- MongoDB 7+
- Redis 7+
- API keys (OpenAI, ElevenLabs, YouTube)

### Installation

```bash
# Clone repository
git clone https://github.com/SultanAiMaster/viralfaceless-ai.git
cd viralfaceless-ai

# Install dependencies
npm install

# Copy environment file
cp .env.example .env

# Edit .env with your API keys
nano .env

# Start Redis & MongoDB
redis-server
mongod

# Start development server
npm run dev
```

### Configuration

Edit `.env` file with your credentials:

```env
# OpenAI API
OPENAI_API_KEY=sk-...

# ElevenLabs API
EOUTEELABS_API_KEY=...

# YouTube OAuth (for each channel)
YOUTUBE_CLIENT_ID=...
YOUTUBE_CLIENT_SECRET=...
CHANNEL_1_REFRESH_TOKEN=...
CHANNEL_2_REFRESH_TOKEN=...
# ... up to CHANNEL_10_REFRESH_TOKEN
```

### Deploy to Production

```bash
# Build TypeScript
npm run build

# Start production server
npm start

# Or use PM2 for process management
pm2 start dist/app.js --name viralfaceless
pm2 startup
pm2 save
```

### Cloud Deployment

**Recommended: Railway.app**
```bash
# Install Railway CLI
npm install -g @railway/cli

# Login
railway login

# Deploy
railway init
railway up
```

**Alternative: DigitalOcean**
```bash
# Create droplet (Ubuntu 22.04)
# SSH into droplet
# Install Node.js, MongoDB, Redis
# Clone repo & follow installation steps
```

## 🔧 Configuration

### Channel Configuration

Edit `src/config/channels.ts` to customize:

```typescript
{
  id: "ch-001",
  name: "WealthMindset",
  niche: "money-wealth",
  categoryId: 22,
  targetAudience: "USA, India, UK",
  uploadTimes: ["08:00", "14:00", "20:00"],
  timezone: "America/New_York",
  language: "en",
  enabled: true,
}
```

### Scheduling

Edit `src/queue/scheduler.ts` to change upload times:

```typescript
cron.schedule("0 8 * * *", async () => {
  // Research task at 8:00 AM
});

cron.schedule("0 9 * * *", async () => {
  // Generation task at 9:00 AM
});
```
